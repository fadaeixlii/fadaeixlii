With over **6 years of experience** building scalable and secure web applications—especially for blockchain and cryptocurrency platforms—I specialize in React, Next.js, Node.js, NestJS, and Solidity. I design clean, reusable systems with a strong focus on performance, PWA features, and seamless integration between front-end and back-end services.
